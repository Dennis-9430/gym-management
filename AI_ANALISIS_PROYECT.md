# 🧠 Rol
Actúa como un **Senior Software Engineer + Tech Lead especializado en React, TypeScript y arquitectura frontend escalable**.

# 🎯 Objetivo
Realizar una **auditoría completa del proyecto** enfocada en:
- detección de errores
- eliminación de código duplicado
- mejora de arquitectura
- optimización de rendimiento
- buenas prácticas

# 📦 Contexto
El proyecto es un sistema de gestión (gym-management) con módulos como:
- ventas
- clientes
- empleados
- suscripciones
- dashboard

Stack:
- React
- TypeScript
- Vite
- CSS modular

# 🔍 Tareas

## 1. 🐛 Detección de errores
- Identificar errores de TypeScript
- Problemas de tipado incorrecto o innecesario
- Imports mal definidos
- Props mal tipadas
- Hooks mal utilizados
- Posibles bugs lógicos

## 2. 🔁 Código duplicado
- Detectar lógica repetida en:
  - componentes
  - hooks
  - utils
- Proponer refactorización:
  - custom hooks
  - funciones reutilizables
  - helpers

## 3. 🏗️ Arquitectura del proyecto
Evaluar:
- Estructura de carpetas
- Separación de responsabilidades
- Acoplamiento entre componentes
- Uso correcto de:
  - hooks
  - context
  - utils

Proponer:
- Nueva estructura si es necesario
- Mejores prácticas escalables

## 4. ⚡ Optimización
- Renderizados innecesarios
- Uso incorrecto de useEffect
- Falta de memoización (useMemo, useCallback)
- Componentes demasiado grandes

## 5. 🧹 Limpieza de código
- Código muerto
- Variables sin uso
- Consoles innecesarios
- Imports duplicados o no usados

## 6. 📐 Buenas prácticas
- Naming conventions
- Organización de tipos
- Separación UI vs lógica
- Manejo de estado
