# Plan de Refactorización del Proyecto Gym Management

## Paso 1: Analizar arquitectura
Analiza completamente la arquitectura del proyecto dentro de la carpeta src.

Identifica:
- estructura de carpetas
- componentes principales
- hooks
- tipos
- utilidades

No modifiques archivos todavía.

---

## Paso 2: Diagnóstico
Explica los problemas de arquitectura encontrados:
- lógica mezclada con UI

- duplicación de código
- problemas de organización

---

## Paso 3: Propuesta de arquitectura
Propón una arquitectura profesional basada en:

- React
- TypeScript
- Clean Architecture

Muestra la nueva estructura de carpetas.
